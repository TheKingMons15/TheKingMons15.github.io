# Fotos de los Integrantes del Equipo

Coloca las fotos de los integrantes en esta carpeta con los siguientes nombres:

- `katty-aguas.jpg` - Foto de Katty Aguas
- `angelo-benavides.jpg` - Foto de Angelo Benavides
- `wladimir-chalcan.jpg` - Foto de Wladimir Chalacán

**Recomendaciones:**
- Formato: JPG o PNG
- Tamaño recomendado: 400x400 píxeles (cuadrado)
- Peso máximo: 500KB por imagen
- Fondo: Preferiblemente uniforme o profesional

Mientras no se agreguen las fotos, se mostrará un icono de usuario por defecto.
